Detailed Description

Handles transition from LASTRADA_PROPERTIES to schema_migrations.

This class provides utilities to migrate from the legacy LUP version tracking (stored in LASTRADA_PROPERTIES table) to the modern schema_migrations table used by Lightweight's migration system.

Initialize transition - call once on first dbtool run.

This method:

  1. Queries LASTRADA_PROPERTIES for current version
  2. Marks all migrations up to that version as applied (in schema_migrations)

If LASTRADA_PROPERTIES doesn't exist or has no version, this is treated as a fresh database and no migrations are marked as applied.

Parameters
managerThe migration manager.
connectionActive database connection.
Returns
true if transition was performed (or not needed), false on error.
